# Натяжные потолки — Калькулятор и Каталог

[![Vite](https://img.shields.io/badge/Vite-646CFF?logo=vite&logoColor=white)](https://vitejs.dev/)
[![React](https://img.shields.io/badge/React-61DAFB?logo=react&logoColor=black)](https://react.dev/)
[![pnpm](https://img.shields.io/badge/pnpm-F69220?logo=pnpm&logoColor=white)](https://pnpm.io/)

Сайт-каталог с калькулятором стоимости натяжных потолков. Проект создан для демонстрации навыков React-разработки.

## 🎯 Функциональность

- **Каталог потолков** — 6 видов: матовые, глянцевые, сатиновые, парящие, двухуровневые, световые линии
- **Калькулятор стоимости** — динамический расчет с учетом:
  - Площади помещения
  - Типа полотна (матовый/глянцевый/цветной/BAUF)
  - Количества светильников и люстр
  - Дополнительных опций (мебель, плитка)
- **Адаптивный дизайн** — корректно отображается на всех устройствах
- **Маршрутизация** — React Router DOM

## 🚀 Технологии

- **React 18** — библиотека для UI
- **Vite** — сборка проекта (в 10-20 раз быстрее CRA)
- **pnpm** — быстрый менеджер пакетов
- **React Router DOM** — маршрутизация
- **CSS Modules** — изолированная стилизация компонентов

## 📦 Установка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/SanyaSkl/ceilings-project.git

# Перейти в папку проекта
cd /ceilings-project

# Установить зависимости (требуется pnpm)
pnpm install

# Запустить dev-сервер
pnpm dev

# Собрать продакшн версию
pnpm build

# Предпросмотр сборки
pnpm preview
